# Asset Generation Prompts (Optional external workflow)
- Since this repo currently uses procedural Canvas art, use these prompts if you want to generate original assets later.
- Output files should be placed under `assets/ships`, `assets/enemies`, `assets/bosses`, `assets/ui`.

## Ship prompt
"Original futuristic top-down fighter jet, clean silhouette, neon accent, no logos, transparent background, game sprite sheet style"

## Enemy prompt
"Original sci-fi drone aircraft top-down, hostile silhouette, readable shape at small sizes, transparent background"

## Boss prompt
"Original massive sci-fi airborne fortress top-down, glowing core weakpoint, layered armor, transparent background"

## Background prompt
"Original high-altitude sci-fi battleground matte, no text, no copyrighted logos"
